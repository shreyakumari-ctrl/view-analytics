# MyTube Studio - Creator Analytics Dashboard

A comprehensive analytics platform for content creators to track views, revenue, subscribers, and audience engagement in real-time. Inspired by YouTube Studio, this project provides creators with powerful insights to optimize their content strategy.

## 🎯 Features

### 📊 Dashboard
- Real-time analytics overview
- Quick stats on views, subscribers, and revenue
- Interactive charts and graphs
- Personalized notifications for important events

### 📈 Analytics
- Detailed view analytics with trends
- Watch time metrics
- Click-through rates (CTR)
- Engagement analytics (likes, comments, shares)

### 👥 Audience
- Viewer demographics
- Geographic distribution
- Audience growth trends
- Subscriber insights

### 💰 Monetization
- Revenue tracking
- CPM and RPM metrics
- Ad performance analytics
- Earnings reports

### 📽️ Videos
- Video performance metrics
- Upload and publish analytics
- Video-specific viewer data
- Content performance comparison

### ⚙️ Settings
- User profile management
- Account preferences
- Theme customization
- Notification preferences

## 🛠️ Technology Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with responsive design
- **JavaScript** - Interactive functionality

### Backend/Database
- **JavaScript** - Client-side logic

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server or backend setup required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/shreyakumari-ctrl/view-analytics.git
cd view-analytics
```

2. Open with a local server (recommended):
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

3. Open in your browser:
```
http://localhost:8000
```

Or simply open `index.html` directly in your browser.

## 📋 Key Metrics Tracked

- **Views**: Total video views
- **Watch Time**: Hours watched by viewers
- **Subscribers**: Channel subscriber count
- **Revenue**: Earnings from monetization
- **Engagement**: Likes, comments, shares
- **Audience Demographics**: Age, gender, location
- **Traffic Sources**: Where viewers come from
- **Device Type**: Desktop, mobile, tablet distribution

## 🔒 Data & Privacy

This is a frontend-focused application primarily designed for demonstration and local use. User data is stored in browser's local storage. For production use, implement proper backend authentication and data security measures.
